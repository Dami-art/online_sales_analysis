
Online Sales Analysis 

Overview

A simple Python OOP project created as part of the final assignment for the Introduction to Git and GitHub module - IT Academy · AI & Python Development · Generation 2025.

The project demonstrates product management, a basic shopping cart system, and proper Git workflow practices.

Components

- product.py – Product class (name, price, quantity)
- product_manager.py – ProductManager (add, list, remove products; inventory value)
- cart.py – Cart (add items, show cart, total price)
- main.py – Demonstrates all functionality
- config.json – Local config file (ignored by Git)
- .gitignore – Excludes config.json and screenshots

Git Workflow

- Repo created on GitHub and cloned locally
- Feature branches used: add-product-removal, add-cart-functionality
- Features implemented and merged back into main
- All changes committed with clear messages and pushed to GitHub

Run

python main.py
